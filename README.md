# MOODBEATS — Landing Page III

MOODBEATS Landing Page III — a clean, responsive landing page built with plain HTML, CSS, and JavaScript.

![MOODBEATS Landing Page Screenshot](./Screenshot%202025-02-25%20111330.png)

## Demo
Open `index.html` in your browser to view the page locally.

## Features
- Modern, responsive layout optimized for desktop and mobile
- Navigation, hero section, features/cards, and contact/CTA area
- Lightweight: no build step or external dependencies required

## Tech
- HTML
- CSS
- JavaScript (vanilla)

## Run locally
1. Clone the repository:

   git clone https://github.com/BinaryVortex/MOODBEATS-Landing-Page-III.git
2. Change into the project directory:

   cd MOODBEATS-Landing-Page-III
3. Open the page in your browser:

   - Double-click `index.html`, or
   - Run a simple HTTP server (optional):
     - Python 3: `python -m http.server 8000` then visit `http://localhost:8000`
